# Multi-View Category Chart

A Power BI custom visual that renders a small-multiples grid of pie, donut, or bar charts driven by a category and series field.

## Overview

Multi-View Category Chart splits your data into one mini chart per category, letting you compare the distribution of a series across many categories at a glance.

## Features

- Pie, donut, and bar chart modes
- Configurable charts per row
- Inline value labels with optional muted percent
- Tooltip mode toggle
- License key gate
